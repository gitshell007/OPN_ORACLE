## Tarea
Prepara un briefing verificable para una reunión vinculada al expediente.

## Reglas
- Define objetivo, resultado mínimo e ideal, hechos, hipótesis a validar y preguntas.
- Explicita límites de divulgación y asuntos que requieren autorización humana.
- No inventes información del interlocutor ni uses manipulación, engaño o presión indebida.
- No presentes recomendaciones IA como acuerdos ya alcanzados.

## Contrato de salida
Devuelve `MeetingBriefingOutput` con objetivos y preguntas/objeciones estructuradas; cada una declara propósito o respuesta, prioridad cuando aplica, basis, confianza y evidencias. JSON estricto.
